## v3.1.5 주요 변경 (← v3.1.4)

| 신규 기능 | 설명 |
|----------|------|
| **PROP CONTINUITY TRACKER** | 영화 산업의 Script Supervisor 역할을 AI에게 부여. 매 씬·매 비트 끝에 추적 대상 소품의 위치·상태를 INTERNAL 메모로 강제 기록. |

| **금지 패턴 6종 명시 차단** | "회수된 소품의 재출현"·"동반 소품의 순간이동"·"시들지 않은 시간 점프"·"보관 소품 위치 점프"·"음식·음료 양 모순"·"위치 망각" 6가지 흔한 실수를 사전 차단. |
| **이전 씬 메모 자동 강조 추출** | 직전 비트의 `[소품 상태]` 메모를 정규식으로 추출해 다음 비트 집필 시 별도 강조 블록으로 주입. AI가 이전 위치를 잊지 않게 한다. |
| **메모 자동 분리** | INTERNAL 메모는 AI 자기 관리용이므로 최종 시나리오 본문에 노출되지 않음. main.py가 DOCX 빌드 시 자동 제거. |
| **모순 발생 시 자가 수정 절차 4종** | 행동 명시 / 시간 단서 / 생략 회피 / 전 비트 수정 — AI가 모순을 감지하면 4가지 중 하나로 즉시 해결. |

## v3.1.4 주요 변경 (← v3.1.3)

| 신규 기능 | 설명 |

| **시간·발신자 표기** | 형식 B에서 시간이나 발신자 자동 포맷 (`[핸드폰 / 카톡 04:13]`, `[강회장 문자]`, `[TV 뉴스 21:00]`). |

## Period Pack 10 시대 상세 (v3.1.2)

비트 집필 시 세계관·로그라인·트리트먼트·스토리요소 텍스트를 스캔해 해당 시대 블록을 자동 재주입한다.

| 시대 | 연도 | 대표 레퍼런스 |
|---|---|---|
| **조선 전기** | 1392~1592 | 뿌리깊은 나무·육룡이 나르샤·용의 눈물 |
| **조선 중기** | 1592~1700 | 명량·최종병기 활·광해·사도·남한산성 |
| **조선 후기** | 1700~1876 | 관상·명당·역린·군도·사바하·조선명탐정 |
| **구한말** | 1876~1910 | 미스터 션샤인·덕혜옹주·귀향 |
| **일제강점기 전기** | 1910~1931 | 밀정·암살·박열·말모이·아가씨·경성스캔들 |
| **일제강점기 후기** | 1931~1945 | 동주·군함도·파친코·귀향·봉오동 전투 |
| **해방정국** | 1945~1950 | 태극기 휘날리며 전반부·국제시장 전반부 |
| **한국전쟁** | 1950~1953 | 태극기 휘날리며·웰컴 투 동막골·고지전·포화 속으로 |
| **개발독재기** | 1960~1987 | 변호인·1987·택시운전사·서울의 봄·실미도·화려한 휴가 |
| **민주화기** | 1987~1999 | 살인의 추억·써니·응답하라 1988·건축학개론 |

**각 시대별 주입 필드 (11개):**
1. 복식 (계급별·성별 의복 디테일)
2. 관직·호칭 (시대별로 다른 호칭 체계)
3. 주거·공간 (거주·업무 공간)
4. 통화·거래 (화폐·경제 방식)
5. 음식·기호품 (외식 문화 포함)
6. 교통 (이동 수단)
7. 언어의 결 (대사 작성 시 톤)
8. 시대 사건·사회 맥락 (정치·사회)
9. **작가 흔한 고증 오류 — 절대 금지** (가장 중요)
10. 핵심 레퍼런스 씬 (실제 영화 디테일 사례)

**자동 감지 알고리즘:**
- 연도 4자리 키워드: 가중치 ×2 (강한 신호)
- 인물·사건·용어 키워드: 가중치 ×1
- 점수 상위 2개 시대까지 주입
- 3개 이상 감지 시 교차 전개 안내문 추가

## Beat 15 엔딩 분기 (v3.1부터)

Creator Engine의 `ending_payoff_type` 값에 따라 엔딩 규칙이 자동 분기된다.

### INTERNAL_TRANSFORMATION (내적 전환형)
예: 쿠킹클래스 — 유진이 분석을 멈추고 세웅의 이름을 먼저 부르는 순간 엔딩.

### EXTERNAL_CHOICE (외적 선택형)
주인공의 명확한 외적 행동으로 엔딩 확정. 장르 약속 준수.

### 미지정 (빈 값)
기존 v3.0 장르 기반 엔딩 규칙으로 폴백.

## PROP CONTINUITY 추적 시스템 (v3.1.5)

영화 산업의 *Script Supervisor*가 매 컷마다 소품 연속성을 추적하듯, AI가 매 씬 끝에 INTERNAL 메모로 소품 위치/상태를 기록한다.

### 추적 등급 3종

| 등급 | 추적 강도 | 예시 |
|------|----------|------|
| **A. 서사 핵심** | 매 씬 추적 | 노트·일기·수첩 / 사진·편지·쪽지 / 반지·시계·목걸이 / 열쇠·통장·계약서 / 무기·증거품·약병 / 부적·USB·녹음기 |
| **B. 캐릭터 동반** | 등장 씬 추적 | 핸드폰·가방·지갑 / 코트·우산·모자 / 책·태블릿·노트북 / 음료·도시락·꽃다발 |
| **C. 환경 변화** | 시간 흐름 따라 추적 | 화환·꽃다발 / 음식·음료 / 시계·달력 / 우편물·서류 더미 / 빨래·설거지 |

### 매 씬 끝 INTERNAL 메모 (예시)

```
[소품 상태 / S#3 종료 시점]
- 노트(세웅): 세웅 가방 안 (수업 중 사용 → 회수)
- 화환: 스튜디오 입구 (시든 채 그대로)
- 카드 두 장(이진호/반세웅): 아일랜드 위
- 핸드폰(유진): 카드 밑 (진동 멎음)
- 사진(런던 X자): 박스 밑 서랍 안 (안 봄)
```

이 메모는:
- AI 자신이 다음 비트 집필 시 *입력 컨텍스트*로 받음
- prompt.py의 `prev_block`이 정규식으로 추출해 강조 블록으로 다시 주입
- **최종 시나리오 본문에는 노출되지 않음** (main.py `_strip_prop_state_memos()`가 자동 제거)

### 금지 패턴 6종 명시 차단

1. **회수된 소품의 재출현** — "가방에 넣고 떠남" → 다음 씬에 "테이블 위" ❌
2. **동반 소품의 순간이동** — "주머니에 넣고 외출" → "아일랜드 위에서 진동" ❌
3. **시들지 않은 채 시간 점프** — "시든 화환" → 한 달 후 "활짝 핀 국화" ❌
4. **위치가 갑자기 바뀐 보관 소품** — "서랍 잠금" → "협탁 위" ❌
5. **음식·음료 양 증감 모순** — "절반 마심" → 3분 뒤 "가득 참" ❌
6. **잠재적 소품의 위치 망각** — "코트 주머니에 넣음" → "코트 없이 외출" → 봉투 어디? ❌

### 모순 발생 시 자가 수정 4종

1. **행동 명시** — 누군가가 옮기는 행동을 씬에 추가
2. **시간 단서** — 누가 옮겼는지 암시 ("다은이 어제 다녀간 자리")
3. **생략 회피** — 모순 자체 제거 ("아일랜드는 비어 있다")
4. **전 비트 수정** — 이전 비트가 잘못된 경우 작가 검토 표시

## INSERT 11종 자동 인식 (v3.1.4)

비트 집필 시 화면에 보이는 모든 텍스트를 표준 표기로 강제한다. AI가 prompt.py의 INSERT 시스템 모듈에 따라 작성하면, main.py가 자동으로 시각 분리.

| 화면 종류 | 권장 형식 | 예시 |
|---|---|---|
| 카톡·메신저 | A 또는 B | `[핸드폰 / 카톡] '아빠: 임대료 30프로 올린다.'` |
| 문자(SMS) | B 또는 C | `[강회장 문자] '임대 조건 재검토.'` |
| 이메일 | A | `INSERT — 노트북 이메일 화면` |
| 유튜브·TV·뉴스 | B | `[TV 뉴스 자막] '한남동 임대료 인상'` |
| SNS·인스타·DM | A 또는 B | `[인스타 / DM] '안녕하세요.'` |
| 검색 화면 | C | `검색창에 친다. '한남동 임대료 시세'.` |
| 노트·일기·메모 | C | `노트 첫 줄. '재료 파악은 요리의 반.'` |
| 신문·기사 | B | `[신문 헤드라인] '한남도 일대 인상.'` |
| 자막 | B 또는 C | `화면 자막: '3개월 후'` |
| 알림 | C | `핸드폰 알림. '예약 1건 / 이진호.'` |

**표기 형식 3종:**
- **형식 A**: `INSERT — [헤더] / 본문 / [/INSERT]` (3줄 이상 긴 메시지)
- **형식 B**: `[라벨] '본문'` (1~2줄 짧은 인서트)
- **형식 C**: 지문 안에 따옴표로 자연스럽게 (1줄)

DOCX 시각 효과: 헤더(작은 굵은 글씨, 들여쓰기 2.55cm) + 본문(이탤릭, 들여쓰기 2.55cm) + 자동 빈 줄 분리.
